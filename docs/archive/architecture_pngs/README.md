# Archived Architecture PNGs

This folder stores static architecture PNGs that were previously kept at the top level of `docs/`.

These files are retained for historical reference and documentation continuity.

Use this folder for:

- legacy architecture snapshots
- static export artifacts from older design iterations
- markdown references that still depend on image-based diagrams

Do not treat these PNGs as the source of truth for current architecture decisions.

Prefer actively maintained documentation in:

- [docs/architecture](/Users/ningli/project/seedcore/docs/architecture)
- [docs/development](/Users/ningli/project/seedcore/docs/development)

If a diagram is still operationally important, prefer replacing it with:

- a maintained markdown document
- a mermaid diagram
- an explicitly versioned source file for the exported image
